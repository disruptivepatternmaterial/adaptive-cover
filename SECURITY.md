# Security Policy

## Reporting a vulnerability

Please do not open public issues for suspected security vulnerabilities.

Instead, report privately to the maintainers via GitHub Security Advisories:

- Go to the repository **Security** tab.
- Click **Report a vulnerability**.

Include:

- affected version(s)
- reproduction steps
- potential impact
- suggested mitigation (if known)

## Disclosure

- We will acknowledge reports and triage as quickly as possible.
- We will coordinate a fix and release before public disclosure when feasible.
